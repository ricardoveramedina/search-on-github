# Search on github

Search for a Repository on github. This project use github REST API with octokit.

## Before starting

Before starting don't forget to add your API key into .env
VITE_GIT_TOKEN='KEY'

[GITHUB DOCUMENTATION](https://docs.github.com/en/rest/guides/getting-started-with-the-rest-api)

Iss possible to request without a key but there is a Rate Limit
[GITHUB SEARCH API](https://docs.github.com/en/rest/search)
